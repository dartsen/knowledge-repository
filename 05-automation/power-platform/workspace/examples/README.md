# Power Platform Workspace Examples

- Copy `example.local-config.json` to `../workspace.local.json` and replace placeholders. Do not add credentials.
- Read `daily-command-sequence.ps1` as an educational sequence. It is intentionally inert unless `-RunExample` is supplied and still omits PAC `-Execute` switches.

Examples are not Production configuration and must not be treated as organization-specific defaults.
