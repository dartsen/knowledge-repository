# Artificial Intelligence

## Scope

Use this domain for AI foundations, generative-AI applications, prompt and context design, retrieval, agents, tools, model evaluation, responsible use, and production operations.

## Recommended Starting Point

Begin with problem framing, data and risk classification, and measurable success criteria. Learn prompting and retrieval before agentic tool use, then progress to evaluation, governance, monitoring, and incident response.

## Key Concepts

- Model capabilities, limits, context, and inference
- Prompting, structured outputs, retrieval, and grounding
- Agents, tools, permissions, and approval boundaries
- Evaluation datasets, quality gates, and regression testing
- Model routing, latency, cost, and capacity
- Prompt injection, data leakage, misuse, and incident response
- Responsible-AI governance and human accountability

## Common Uses

- Evaluate whether AI is appropriate for a business problem.
- Design a grounded assistant or RAG workflow.
- Build a controlled agent that calls tools.
- Establish production evaluation and monitoring.
- Review an AI solution for risk, cost, and operational readiness.
