# Change Management Governance Checklist

## Ownership

- [ ] Business owner
- [ ] Technical owner
- [ ] Support owner
- [ ] Escalation path

## Access and Security

- [ ] Least privilege
- [ ] Sensitive data reviewed
- [ ] Access review cadence

## Change Management

- [ ] Impact assessed
- [ ] Testing evidence
- [ ] Approval captured
- [ ] Rollback or remediation plan

## Auditability

- [ ] Version history
- [ ] Decision record
- [ ] Logs or evidence
- [ ] Documentation stored

## Operations

- [ ] Monitoring
- [ ] Runbook
- [ ] Incident process
- [ ] Review cadence
