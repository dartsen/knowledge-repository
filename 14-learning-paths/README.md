# Learning Paths

Learning paths organize existing canonical documents from foundations through architecture, implementation, operations, and governance. They do not copy the main guides.

## Progression

```text
Orientation → Fundamentals → Architecture → Patterns
→ Implementation → Operations → Governance → Advanced practice
```

Choose a domain path when learning a discipline. Choose a cross-domain path when preparing for a role or delivering a workload that spans multiple technologies.
