# AI Agents Beginner-to-Pro Learning Path

## 30-Day Plan

- Learn vocabulary.
- Read quick reference.
- Map a simple scenario.
- Create a checklist.

## 60-Day Plan

- Apply to a realistic work example.
- Document risks and governance.
- Create troubleshooting notes.

## 90-Day Plan

- Create a reusable pattern.
- Review with a stakeholder.
- Add lessons learned.

## Levels

| Level | Focus | Artifact |
|---|---|---|
| Beginner | Understand | Summary |
| Advanced Beginner | Follow | Checklist |
| Intermediate | Apply | Example |
| Advanced | Improve | Runbook |
| Enterprise Professional | Govern | Standard |
| Architect / Strategic Lead | Lead | Roadmap |
