# Review Required

This area preserves content whose accuracy, authority, ownership, placement, scope, or publication status cannot be determined safely.

## Review Outcomes

- Promote to Draft or Active in a primary domain.
- Merge unique content into a canonical document and archive the source as superseded.
- Move to organization-specific Project Delivery guidance.
- Archive as duplicate, obsolete, incomplete, or historical.

The `generated-scaffolding/` hierarchy retains original subject context while repetitive drafts are curated. Review-required content is excluded from normal recommendations.
