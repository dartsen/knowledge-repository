# Archive

The archive preserves non-active content without deleting history.

## Reasons

- `superseded/`: replaced by a named canonical document
- `duplicates/`: exact or normalized duplicate copies
- `obsolete/`: no longer applicable to supported technology or process
- `historical-projects/`: retained project evidence
- `historical-generation/`: manifests and artifacts from repository-generation phases
- `incomplete/`: abandoned or empty material with no active value

Archived documents should retain useful metadata, state their archive reason, and link to a replacement when one exists. Age alone is not an archive reason.
