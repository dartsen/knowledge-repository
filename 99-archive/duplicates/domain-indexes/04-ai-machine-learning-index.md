# AI and Machine Learning

This folder contains expanded beginner-to-pro guides for the **AI and Machine Learning** domain.

## Topics

| Topic | Purpose |
|---|---|
| [AI Fundamentals](../../../06-artificial-intelligence/ai-fundamentals/README.md) | AI Fundamentals covers the basic ideas behind artificial intelligence, machine learning, generative AI, and how these capabilities are applied responsibly in business. |
| [Prompt Engineering](../../../06-artificial-intelligence/prompt-engineering/README.md) | Prompt Engineering is the practice of giving AI systems clear instructions, context, constraints, examples, and evaluation criteria so they produce more useful outputs. |
| [Retrieval-Augmented Generation](../../../06-artificial-intelligence/rag/README.md) | Retrieval-Augmented Generation, or RAG, combines AI generation with external knowledge retrieval so responses can be grounded in approved sources. |
| [AI Agents](../../../06-artificial-intelligence/ai-agents/README.md) | AI Agents are AI-powered systems that can plan, call tools, use memory or context, and take multi-step actions toward a goal under defined controls. |
| [Model Evaluation](../../../06-artificial-intelligence/model-evaluation/README.md) | Model Evaluation is the process of measuring whether an AI or machine learning model is accurate, safe, useful, reliable, and fit for purpose. |
| [Responsible AI](../../../06-artificial-intelligence/responsible-ai/README.md) | Responsible AI is the discipline of designing, using, monitoring, and governing AI systems in ways that are safe, fair, transparent, accountable, and human-centered. |
| [AI Governance](../../superseded/ai-domain-ai-governance/README.md) | AI Governance defines the policies, controls, roles, approval gates, monitoring, and accountability needed to manage AI use responsibly. |
| [Applied AI Patterns](../../../06-artificial-intelligence/applied-ai-patterns/README.md) | Applied AI Patterns are reusable ways to apply AI to common work such as summarization, classification, extraction, drafting, analysis, search, and decision support. |
